# Video Game Market Analysis

Exploratory data analysis project focused on identifying success patterns in global video game sales.

## Business Objective

Support strategic marketing decisions for 2017 using historical data up to 2016.

## Dataset

The dataset includes:

- Game titles
- Platforms
- Release year
- Genre
- Regional sales
- Critic scores
- User scores
- ESRB ratings

## Key Questions

- Which platforms are growing or declining?
- Which genres perform best?
- Do reviews influence sales?
- Are regional preferences different?
- How does age rating impact sales?

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
